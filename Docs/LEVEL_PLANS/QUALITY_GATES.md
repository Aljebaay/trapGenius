# Quality Gates (Must Pass)

## Gate A — Readability
- Player can explain why they died after 1 retry.
- No unsignaled instant kills (unless explicitly labeled RNG level).

## Gate B — One Betrayal
- Exactly one primary deception per level.
- Support is mechanical only.

## Gate C — Retry Speed
- From death to movement: target < 1.5s (hard max 2.5s).

## Gate D — Solve Time
- Known solution run: 5–10 seconds.

## Gate E — Variety
- No same primary pattern within last 5 levels.
- No same expectation within last 3 levels.
